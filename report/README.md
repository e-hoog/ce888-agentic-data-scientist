## Report guidance (suggested structure)

1. Introduction (goal, why agentic)
2. System architecture (agents/tools/data flow)
3. Dataset understanding (signals, why they matter)
4. Planning logic (conditional decisions)
5. Tool use: modelling and evaluation (metrics, baselines)
6. Reflection and re-planning (what triggers changes; what changes)
7. Memory & learning (what is stored; how it is used)
8. Ethics & limitations
9. Conclusion & future work
